# Weather Diary

## Автор
Айсылу Фахертдинова

## Описание программы
Приложение для ведения дневника погоды с возможностью добавления, просмотра и фильтрации записей. Данные сохраняются в формате JSON.

## Как запустить приложение
1. Установите Python 3.x.
2. Установите зависимости:
   pip install -r requirements.txt
3. Запустите:
   python main.py

## Примеры использования
1. Введите дату, температуру, описание и отметьте осадки.
2. Нажмите «Добавить запись».
3. Используйте фильтры по дате и температуре.
4. Сохраняйте и загружайте данные через меню «Файл».
